# Data

This folder contains fictional datasets created for the **Warehouse KPI Dashboard** project.

The datasets are designed to simulate warehouse operations for a fictional meal-kit distribution center and are used for analytics, reporting, and KPI calculations.

## Datasets

| File | Description |
|------|-------------|
| inventory.csv | Current warehouse inventory by SKU and storage location |
| orders.csv | Customer order transactions and fulfillment data |
| receipts.csv | Supplier receipts and putaway activities |
| cycle_counts.csv | Physical inventory count results used to measure inventory accuracy |

## Notes

- All data in this project is fictional.
- The datasets are designed for learning, portfolio demonstration, and warehouse analytics.
- No confidential or proprietary business information is included.
